连接召唤、同调召唤、超量召唤等不入连锁的召唤操作不会转移优先权。

以镇魂棺与尼比鲁为例：
回合玩家进行连接召唤后，优先权依然在回合玩家手中。
回合玩家可以自己C1发动镇魂棺的②效果（二速效果），也可以选择不发。
对方不能C1发动尼比鲁，必须等回合玩家放弃优先权后才能发动。

```
1. 优先权仅在两种情况下转移：
   - 发动效果时（连锁开始构建，优先权转移给对方以确认是否连锁）
   - 回合玩家宣言进入下一阶段时（或主动放弃优先权）

2. 以下操作不转移优先权：
   - 召唤（通常召唤、特殊召唤、连接召唤、同调召唤、超量召唤等）
   - 盖放卡片
   - 攻击宣言
   - 不入连锁的效果处理或规则动作
```

白银姬C1大欢迎C2自排连锁防灰的操作是错误的——因为发动效果时优先权已转移给对方，对方有权选择是否连锁。

结束阶段的优先权

结束阶段中回合玩家持有优先权，进行三选一：
- 处理一个已触发的效果
- 发动一个效果
- 放弃优先权（让对方进行三选一）

```
1. 回合玩家先进行三选一。若选择放弃，非回合玩家进行三选一。

2. 若双方都放弃优先权：
   - 场上无必须处理/发动的效果 → 回合结束
   - 场上有必须处理/发动的效果 → 优先权回到有必发效果的玩家，此时只有二选一（处理或发动，无放弃选项）

3. 任一方选择了处理或发动之后 → 快速时点 → 优先权回到回合玩家 → 重新三选一。如此循环。
```

以冰剑龙与耀变为例：
我方冰剑龙因对方耀变效果被破坏，冰剑龙③效果在结束阶段触发（选发）。对方耀变在结束阶段从除外区回归（必发）。我方可以先放弃优先权，等对方必须处理耀变回归后，优先权回到我方，再处理冰剑龙的炸卡效果。因此可以在耀变回归之后再将其炸掉。
