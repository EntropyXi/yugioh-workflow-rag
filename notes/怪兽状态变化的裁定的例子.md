冰剑龙②效果的"下个回合此卡不可使用此效果"是效果处理时对自身适用的附加限制。该限制是否在翻面后保留，取决于冰剑龙处理效果时自身的表侧/里侧状态。

以C1冰剑龙②效果、C2日全食之书为例：

```
1. C1冰剑龙②效果 → C2日全食之书
   冰剑龙处理C1时自身已是里侧表示（被C2盖放）。
   限制在里侧状态下被适用 → 结束阶段翻开（里→表）信息不丢失 → 下回合不可使用②效果。

2. 冰剑龙先表侧处理完②效果 → 然后另开连锁C1日全食之书
   冰剑龙处理效果时自身是表侧表示。
   限制在表侧状态下被适用 → 日全食盖放（表→里）信息丢失 → 下回合可以使用②效果。
```

核心原则：
- 表侧→里侧：已适用的效果/限制信息丢失
- 里侧→表侧：已适用的效果/限制信息保留

类似的还有无底的落穴——先把怪兽变里侧再适用"不能变更表示形式"，所以即使之后被翻开，该限制仍有效。
